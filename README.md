# 🍽 Food Hub - Restaurant Ordering Website

## 📌 Project Overview

Food Hub is a responsive restaurant food ordering website built using HTML, CSS, and JavaScript. Users can browse food items, search dishes, add products to a shopping cart, manage quantities, and place orders through a simple checkout process.

The website provides a modern UI with an interactive cart system and responsive design for mobile and desktop devices.

---

## 🚀 Features

### 🔍 Food Search

* Search menu items instantly.
* Displays "Food Not Available" when no matching item is found.

### 🍱 Menu Display

* Multiple food categories including:

  * Andhra Specials
  * South Indian
  * Chinese
  * American
  * Desserts
  * Healthy Foods
* Displays food name, category, description, and price.

### 🛒 Shopping Cart

* Add food items to cart.
* Increase or decrease item quantity.
* Automatic cart count update.
* Dynamic total price calculation.

### 🗑 Cart Management

* Remove items by decreasing quantity to zero.
* Clear all cart items with one click.

### 💳 Checkout System

* Collects:

  * Customer Name
  * Mobile Number
  * Address
* Shows payment success message with total amount.

### 📱 Responsive Design

* Mobile-friendly layout.
* Adaptive cart panel.
* Responsive menu grid and hero section.

## ⚙️ How to Run

1. Download the project files.
2. Keep all files in the same folder:

   * food.html
   * food.css
   * food.js
3. Open `food.html` in any modern web browser.
4. Start browsing and ordering food.
